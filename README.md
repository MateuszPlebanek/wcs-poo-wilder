# Wilder POO Challenge 

Small Java project to practice OOP basics.

## Files
- `Wilder.java`: model of a wilder (firstname + aware) with getters/setters and `whoAmI()`
- `Classroom.java`: creates a few `Wilder` objects and prints the result of `whoAmI()`

## How to run
Run `Classroom.java` as a **Java Applications** (Eclipse).

## Expected output (example)
```text
Je m'appelle Mateusz et je suis aware
Je m'appelle Jeremy et je ne suis pas aware
Je m'appelle Orelia et je suis aware
